Saya telah membangun beberapa aplikasi berbasis desktop dengan menggunakan Python dan framework seperti Tkinter dan PyQt untuk menciptakan antarmuka pengguna yang interaktif dan fungsional. Salah satu proyek saya adalah aplikasi manajemen data yang memungkinkan pengguna untuk menyimpan, mengelola, dan menampilkan informasi dengan antarmuka yang sederhana namun efektif. Saya juga terbiasa mengintegrasikan aplikasi desktop dengan database SQLite dan MySQL, serta mengimplementasikan fitur tambahan seperti laporan PDF dan otomatisasi proses tertentu.
